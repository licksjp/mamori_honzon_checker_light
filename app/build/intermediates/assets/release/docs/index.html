このアプリについて

このアプリは、名前を入力して、姓名判断の目線で吉凶を見るアプリです。吉凶は、１０種類あり、良い方は効果音でピンポーンと鳴り、悪いときは効果音でブーというと音が出ます。音を聞いて、吉凶を判断して下さい。


1.姓名判断について

姓名判断の吉凶の種類

姓名判断の吉凶の表は以下の通りです
(吉凶は全部で10種類あります)

姓名判断の吉凶表

　　 　　
番号良い運気 音の種類 番号 悪い運気 音の種類
No.1 スーパーウルトラ大吉 ◯ No.6 半吉 ☓
No.2 ウルトラ大吉 ◯ No.7 半凶 ☓
No.3 カリスマ大吉 ◯ No.8 凶 ☓
No.4 大吉 ◯ No.9 大凶 ☓
No.5 吉 ◯ No.10 スーパーウルトラ大凶 ☓


吉凶に対して2種類のどっちかの音がなります

◯:ピンポーンと言う効果音が出ます
☓:ブーという効果音が出ます
